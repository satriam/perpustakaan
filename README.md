# perpustakaan
Perpustakaan App Menggunakan CI dan SbAdmin
## Fitur
* CRUD Anggota
* CRUD Buku dan Kategori Buku
* Peminjaman
* Pengaturan Perpustakaan
* CRUD Pengguna
## Dibuat Dengan
* Codeigniter 3
* Bootstrap 4
* SbAdmin2
* Dll
